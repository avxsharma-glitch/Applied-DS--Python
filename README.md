# 📊 Applied Data Analysis & Mathematical Computing

This repository contains practical implementations of data analysis, statistical methods, visualization, network analysis, and linear algebra concepts using Python.

It serves as a learning project and portfolio demonstration of analytical and computational skills.

---

## 🚀 Project Overview

This repository includes:

- 📈 Data Analysis using **pandas**
- 🔢 Numerical Computing with **NumPy**
- 📊 Statistical Analysis using **scipy.stats**
- 📉 Data Visualization with **Matplotlib**
- 🌐 Network Analysis using **NetworkX**
- ➗ Vector Operations & Orthogonality (Linear Algebra)

---

## 📂 Repository Structure

### 🔹 Analysis_Script_(Binder).ipynb
Data analysis notebook using pandas, numpy, scipy.stats, and matplotlib.  
Configured to run in a Binder environment.

### 🔹 Anylysis_py.ipynb
Exploratory data analysis using multiple Python libraries.

### 🔹 Network_X.ipynb
Graph creation, visualization, and network analysis using NetworkX.

### 🔹 Vector.ipynb
Implementation of vector operations and orthogonality concepts.

## ▶️ How to Run

1. Clone the repository:

   git clone https://github.com/avxsharma-glitch/Applied-DS--Python.git

2. Navigate to the project folder:

   cd your-repo-name

3. Install required libraries:

   pip install pandas numpy scipy matplotlib networkx

4. Launch Jupyter Notebook:

   jupyter notebook
